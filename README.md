# robustness

Experiment on E2E network on pascal voc dataset

## Installation

install pytorch
```bash
pip install torch torchvision torchaudio
```

then run
```bash
pip install requirement.txt
```

## Usage

```python
python E2E_pascalvoc.py --root_dir=ROOT_FOR_DATASET --save_dir=ROOT_TO_SAVE_RESULT
```
